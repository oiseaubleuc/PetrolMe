# Petrolisation OMS

**Operations Management System for the Oil & Gas industry**  
Developed by [Hohosolutions](https://hohosolutions.com)

Petrolisation OMS is a premium enterprise demo platform that centralizes production, assets, maintenance, HSE, logistics, inventory and business intelligence for upstream and midstream operators.

## Stack

- Next.js 14 (App Router)
- React 18 · TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- Recharts
- Lucide Icons

## Getting started

```bash
cd petrolisation-oms
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
